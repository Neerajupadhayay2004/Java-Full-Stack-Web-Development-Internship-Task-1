## 📌 Project Overview
This project demonstrates a complete **Full Stack Web Application** using:

- 🌐 Frontend: HTML + CSS
- ☕ Backend: Java Spring Boot
- 🔗 Integration: HTTP POST Request

The frontend collects user data through a contact form and sends it to a Spring Boot backend, which processes and prints the data in the console.

---

## 🏗️ Project Structure

```

landing-page-project
│
├── frontend/
│   ├── index.html
│   └── style.css
│
└── backend/
├── src/
├── pom.xml
└── mvnw

```

---

## 🌐 Frontend Features

- Modern responsive landing page
- Navigation bar
- Hero section
- About section
- Contact form (Name, Email, Message)
- Styled using CSS (gradient UI + responsive design)

---

## ☕ Backend Features

- Spring Boot REST API
- POST endpoint: `/contact`
- Handles form submission
- Prints submitted data in console
- Runs on port `8080`

---

## 🔄 Full Stack Workflow

1. User fills the contact form.
2. Form sends a POST request to:
```

[http://localhost:8080/contact](http://localhost:8080/contact)

````
3. Spring Boot controller receives data.
4. Data is processed and printed in terminal.

## ▶️ How to Run the Project

### 🔹 1. Run Backend

```bash
cd backend
./mvnw spring-boot:run
````

Server will start at:

```
http://localhost:8080
```

---

### 🔹 2. Open Frontend

Open:

```
frontend/index.html
```

Fill the form and submit.

Check backend terminal for printed data.
